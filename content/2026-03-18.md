# 動機

ホームページをリニューアルする際に、`React` エコシステムから脱却して、`11ty` と、素の `HTML (Liquid)` や `CSS` に寄せました。

何が実装されているのかを理解する上で必要な知識を書き記します。今後、HTML を人間が書くということはほとんどないのでしょうけれど...自分のホームページなのにどう作られているかよく分かっていないのは癪なので。

# 11ty

[`eleventy.config.js`](https://www.11ty.dev/docs/config/) このファイルが 11ty プロジェクトの設定を担っています。

```js
return {
  dir: {
    input: "src",
    includes: "_includes",
    data: "_data",
    output: "_site",
  },
};
```

細かい部分は省きますが、どのフォルダをどう扱うかというのを定義しています。

- input: ここがトップレベルのディレクトリとして認識されます。この配下のファイルはテンプレートファイルとして、HTML への変換対象となります。
- includes: テンプレートファイルから参照するファイルとして認識されます。ここに置いたファイルは単体ではテンプレートファイルとしては認識されません。
- data: 各種テンプレートから参照可能なグローバルデータとして認識されます。
- output: テンプレートエンジンによって変換された HTML の出力先として認識されます。
